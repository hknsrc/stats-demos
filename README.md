# Statistical Visualization Demos

This repository contains a set of small interactive visualizations designed to help students develop intuition about key concepts in quantitative methods and factor analysis.

The demos aim to make abstract statistical ideas more visually intuitive and easier to understand in teaching contexts.

## Interactive demos

The current visualizations include:

- **Rotation Intuition** – A visual explanation of factor rotation (unrotated, orthogonal, and oblique solutions).
- **EFA Motion Demo** – An animated illustration showing how factors can emerge from patterns of shared movement.
- **ESEM Visualization / Mini Game** – An interactive example illustrating differences between CFA (ICM) and ESEM when cross-loadings are present.

## Live demo

You can access the interactive demos here:

https://hknsrc.github.io/stats-demos

## Purpose

These visualizations were created purely for **educational purposes** to support teaching and learning in courses related to:

- quantitative methods
- statistics
- factor analysis
- psychometrics
